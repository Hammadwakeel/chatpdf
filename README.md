Full Stack AI ChatPDF Ecosystem

A comprehensive AI platform for document analysis, comparison, and Urdu OCR.

## 📂 Project Structure
- **/backend**: FastAPI server handling LangGraph, RAG pipelines, and vector embeddings.
- **/rag-frontend**: Laravel-based dashboard providing the user interface and service orchestration.

## 🚀 Quick Start (Automated)
Run the master setup script from the root directory:
\`\`\`bash
bash setup_all.sh
\`\`\`

## 🛠️ Individual Setup
If you prefer manual setup:
1. **Backend**: Navigate to \`/backend\`, create a virtual environment, and install \`requirements.txt\`.
2. **Frontend**: Navigate to \`/rag-frontend\`, run \`composer install\`, and setup \`.env\`.
