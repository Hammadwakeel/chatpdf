# Quanti Axionix - Document Intelligence Dashboard

This is the interactive frontend for Quanti Axionix. It provides a seamless, high-performance interface for interacting with document-based AI, featuring specialized modes for standard Q&A, multi-document comparison, and Urdu OCR.

## 🎨 The Tech Stack
* **Framework**: Laravel 11 (PHP)
* **Reactivity**: Alpine.js (Lightweight JavaScript orchestration)
* **Styling**: Tailwind CSS (Utility-first UI design)
* **Communication**: Axios (Bridge to FastAPI)
* **Document Rendering**: Markdown (via Marked.js) & Native PDF iFrames

---

## 🚀 Key Features

### 1. Multi-Mode Interaction
The dashboard supports three distinct operational modes:
* **Q&A Mode**: Deep-dive interaction with a single document.
* **Comparison Mode**: Split-screen interface for analyzing similarities and contradictions between up to 5 documents.
* **OCR Mode**: Specialized pipeline for processing Urdu-scanned images or unreadable PDFs before RAG ingestion.

### 2. Side-by-Side PDF Viewer
A custom-built modal that dynamically adjusts its layout. In Comparison Mode, the dashboard renders multiple PDFs side-by-side using Flexbox, allowing users to verify AI claims against the original source documents in real-time.

### 3. AI-Driven Suggestions
Upon document ingestion, the frontend displays interactive "Suggested Question" chips. These are pre-generated by the Llama 3.3 model, providing users with immediate entry points into the document's content.

### 4. Hybrid Authentication
The frontend manages local Laravel sessions while acting as a secure bearer for the FastAPI JWT tokens, ensuring a unified security layer across both the PHP and Python environments.

## ⚙️ Setup & Installation

### 1. Requirements
* PHP 8.2+
* Composer
* Node.js & NPM

### 2. Environment Configuration
Create a \`.env\` file:
\`\`\`env
FASTAPI_URL=http://127.0.0.1:8000
OCR_API_URL=https://your-ocr-worker.hf.space/upload
DB_CONNECTION=sqlite # Or your preferred DB
\`\`\`

### 3. Install & Build
\`\`\`bash
composer install
npm install && npm run build
php artisan key:generate
php artisan migrate
\`\`\`

### 4. Start the Dashboard
\`\`\`bash
php artisan serve --port=8001
\`\`\`

## 📡 Backend Integration
This frontend acts as a "thick client." Most heavy lifting is delegated to the FastAPI backend.
* **Chat Sync**: Alpine.js polls for real-time AI responses.
* **File Buffering**: PDFs are uploaded to the frontend, proxied to the backend, and then served back via a static mount for viewing.
